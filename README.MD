Library for working with text in various encodings. Currently supported
encodings:

* UTF-8
* UCS-4

#### Building
CMake required
```
$ mkdir build
$ cd build
$ cmake ..
$ make
```

#### Installing
CMake required
```
$ cd build
$ make install
```

#### Running test
```
$ cd build
$ ./test_libuc
```

#### Usage
Look include/libucinvert.h
