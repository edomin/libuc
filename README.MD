Library for converting text between various encodings. Currently supported
conversions:

* UTF-8 -> UCS-4
* UCS-4 -> UTF-8

#### Building
CMake required
```
$ make
```
or
```
$ mkdir build
$ cd build
$ cmake ..
```

#### Build and run test
```
$ make run_test
```

#### Usage
Look include/libucinvert.h
